# GOLEM

GOLEM is a tool for visualizing the distribution of **G**ene regulat**O**ry e**LEM**ents within the plant promoters with a focus on male gametophyte.

For more information about the tool itself, refer to the [documentation](./docs/README.md).

## Contact

Users with genomes not yet available in GOLEM or with private transcriptome datasets (e.g., tissue-specific, stress-affected, or mutant samples) can visualize motifs using a private-access version of the software. Contact: [schpetra@sci.muni.cz](mailto:schpetra@sci.muni.cz).
